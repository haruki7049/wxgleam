# simple_text_ctrl

A simple example demonstrating the WxTextCtrl widget in the wx_gleam library.

## What This Example Shows

- Creating a wxWidgets application with a text input control
- Using the `with_text_ctrl()` convenience function
- Basic window management with close event handling

## Running

```sh
gleam run
```

This will open a window with a text input field where you can type text.
Close the window to exit the application.
